# 文思伴学 AI

## 一、项目介绍

文思伴学 AI 是一款面向高中语文学习场景的 Windows 桌面端智能写作助手，聚焦议论文写作中的审题、构思、论证、素材积累和作文修改。

项目面向高中学生和语文教师，核心价值是把大模型能力转化为结构化、可复盘、可收藏、可导出的学习辅导流程。应用强调辅助学习而非直接代写：AI 提供思路、结构、素材方向、片段优化和练习建议，完整表达仍鼓励学生自己完成。

## 二、功能说明

- 审题立意：分析作文题目或材料，提取关键词、核心命题、推荐立意、写作角度和避坑提醒。
- 论点生成：围绕主题生成中心论点、分论点结构、题目分析和素材方向，支持继续深化、换一组论点、生成大纲等二次优化。
- 素材推荐：根据主题推荐人物事例、时评热点、名言警句和使用示范；内置本地作文素材库，无网络也可稳定展示素材。
- 作文诊断：按 60 分制给出综合评分、维度评分、主要问题、修改建议、片段优化示例和下一步练习建议。
- 五步写作助手：串联审题分析、推荐立意、论点结构、素材匹配和写作大纲，一次完成议论文写作思路搭建。
- 历史记录：保存生成和优化结果，支持查看、复制、删除、搜索和筛选；演示模式结果会标记“演示”。
- 收藏夹：收藏好论点、素材、名言、优化片段和立意，支持搜索、类型筛选、复制、删除和导出。
- 演示模式：默认开启，无网络或无 API Key 时也能完整演示五个生成流程。
- API 配置：支持 OpenAI 兼容 Chat Completions 接口，提供 OpenAI、DeepSeek、通义千问、智谱 GLM、Kimi 和自定义接口预设。
- 结果操作：生成结果支持复制，并可导出 TXT、Markdown、HTML；HTML 为离线静态报告，可直接用浏览器打开。

## 三、技术栈

- Electron：桌面端运行与本地能力集成。
- React：前端界面与交互。
- TypeScript：类型安全与结构化结果定义。
- Vite：前端开发与构建。
- electron-builder：Windows 安装包和便携版打包。
- OpenAI 兼容接口：通过 `/chat/completions` 调用用户配置的大模型服务商。

## 四、本地开发

安装依赖：

```bash
npm install
```

启动桌面应用开发环境：

```bash
npm run dev
```

该命令会先编译 Electron 主进程，再启动 Vite 开发服务器，并打开 Electron 桌面窗口。默认开发服务地址为：

```text
http://127.0.0.1:5173
```

构建前端和 Electron 主进程：

```bash
npm run build
```

## 五、打包 exe

打包 Windows 安装包和便携版：

```bash
npm run dist
```

打包产物输出到 `release/` 目录，通常包含：

- `文思伴学 AI-0.0.0-Setup.exe`：安装版。
- `文思伴学 AI-0.0.0-Portable.exe`：便携版，适合比赛现场直接运行或提交。
- `win-unpacked/`：解包后的应用目录，用于本地排查。

建议比赛提交时附带便携版、安装版和本 README，方便评委在不同电脑上快速体验。

## 六、API 配置说明

本应用支持 OpenAI 兼容 Chat Completions 接口。请确认你的服务商支持以下格式：

```text
POST {API Base URL}/chat/completions
```

设置页支持服务商预设，会自动填充 API Base URL，并提供常用模型名称。选择“自定义 OpenAI 兼容接口”时，可自由填写 Base URL 和模型名称。

API Key 仅存储在用户本机，不会上传至本应用服务器。桌面端优先通过 Electron 主进程侧本地存储保存配置；应用只会在用户主动使用真实 API 模式时，把 API Key 用于请求用户配置的大模型服务商。代码中不应在 console 中打印 API Key。

如果不想配置 API Key，可以保持“演示模式”开启，完整体验审题立意、论点生成、素材推荐、作文诊断和五步写作助手。

## 七、比赛演示建议

推荐演示流程：

1. 先在设置页确认“演示模式”开启，确保无网络或无 API Key 时也能稳定展示。
2. 从首页进入，展示整体功能入口、模式状态和新手引导。
3. 打开“作品介绍”，用作答辩时的产品定位、技术架构和教育价值说明。
4. 重点展示“五步写作助手”，输入作文材料后一键生成审题、立意、论点、素材和大纲。
5. 展示“作文诊断”，说明 60 分制评分、维度进度条、主要问题和片段优化建议。
6. 展示“素材推荐”和本地素材库，突出离线素材能力。
7. 演示收藏夹：收藏一个论点、一个素材或一句名言，再进入收藏夹查看。
8. 演示导出：将结果导出为 Markdown 或 HTML，说明便于老师查看和比赛提交。

如果现场已经配置有效 API Key，可关闭演示模式，展示真实 API 模式；若网络不稳定，建议优先使用演示模式完成主流程。

## 八、教育价值说明

文思伴学 AI 的定位是学习辅导工具，而不是代写工具。

- 辅助学生提升审题准确性，减少偏题和误读材料。
- 帮助学生训练论证逻辑，建立中心论点、分论点和素材之间的支撑关系。
- 丰富作文素材积累，降低临场写作时“无话可说”的压力。
- 通过作文诊断帮助学生发现结构、论证、语言和素材运用中的问题。
- 通过“继续思考”和“下一步练习建议”引导学生自主修改、自主表达、持续提升。
